ENGR-UH 3510: Data Structures and Algorithms - Fall 2017

Undergraduate Course, Abu Dhabi Campus

Fall 2017



As applications are getting complex and data rich, there are three common problems that applications face now-a-days. Data Search − Consider an inventory of 1 million items of a store. If the application is to search an item, it has to search an item in 1 million items every time slowing down the search. As data grows, search will become slower. Processor speed − Processor speed although being very high, falls limited if the data grows to billion records. Multiple requests − As thousands of users can search data simultaneously on a web server, even the fast server fails while searching the data. To solve the above-mentioned problems, data structures come to rescue. Data can be organized in a data structure in such a way that all items may not be required to be searched, and the required data can be searched almost instantly.

Course Overview

This course introduces students to the concepts of data structures and algorithms. A low-level programing language is used to illustrate the course topics, which may be C/C++ or a similar programming language. The course covers topics data structures including lists, stacks, queues, trees and graphs, and algorithms analysis.

Intended Learning Outcomes:


  • Describe the fundamental concepts of data structures and algorithms such as behaviors of basic data structures, time complexity, functions, arrays, pointers, reference and class data members and function member;
  • Analyze and solve the basic data sorting and searching problem using programming language;
  • Apply knowledge of data structures and algorithms to analyze and model engineering problems to meet desired needs;
  • Identify the bugs in source code of algorithm design and develop trouble-shooting skills;
  • Be able to understand medium-scale algorithms and find efficient ways to implement them to solve engineering problems.

Prerequisites: Recommended: ENGR-AD-202 Computer System Programming

Grading

Teaching and Learning Methodologies

Students are expected to arrive at class with an understanding of the basic definitions, concepts, and applications of relevant topics. Class time will be devoted to in-depth discussions of course topics. Homework assignments are dedicated for reinforcing course topics via solving representative problem sets and holding class discussions.

Grading

Grades will be based on three computer programming assignments, midterm exam (paper and computer based) and comprehensive Final exam (paper and computer based). The distribution of grades is subject to some revision at the discretion of the instructor. Typical weighting values are given below:

Projects5% * 6
Midterm15% * 3
Final25% * 1

Projects

Project 1

(1) Find the number of pairs with given sum in the array. (2) Detect cycle in a single linked list. project_one.pdf.

Project 2

Reverse a stack using recursion. project_two.pdf.

Project 3

(1) Check if given binary search tree is Height Balanced or not. (2) Convert an unbalanced binary search tree into a balanced tree. project_three.pdf.

Project 4

(1) Build a min heap and a max heap for given array. (2) Heap sort. (3) Find the kth largest element. project_four.pdf.

Project 5

Sort elements by their frequency and index using Hash Table. project_hashtable.pdf.

Project 4

Check if an undirected graph contains cycle or not. project_graph.pdf.
