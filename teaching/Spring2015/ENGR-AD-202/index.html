Course Overview

This course introduces students to the concepts of computer system programming language. A low-level programing language is used to illustrate the course topics, which may be C/C++ or a similar programming language. The topics covered include basic if statements, loops, functions, arrays, strings, pointers, recursion, C++ class and objects, C++ class overloading and class template, C++ inheritance and polymorphism and C++ Files and Streams.

Intended Learning Outcomes:


  • Describe the fundamental programming concepts of variables, functions, arrays, pointers, reference and class data members and function member;
  • Analyze and solve the basic data sorting and searching problem using programming language;
  • Apply knowledge of C++ programming to analyze and model engineering problems to meet desired needs;
  • Identify the bugs in source code and develop trouble-shooting skills;
  • Be able to understand basic algorithms and find efficient ways to implement them to solve engineering problems.

Prerequisites: Recommended: CS-AD 101 Introduction to Computer Science

Grading

Teaching and Learning Methodologies

Students are expected to arrive at class with an understanding of the basic definitions, concepts, and applications of relevant topics. Class time will be devoted to indepth discussions of course topics. Homework assignments are dedicated for reinforcing course topics via solving representative problem sets and holding class discussions.

Grading

Grades will be based on three computer programming assignments, midterm exam (paper and computer based) and comprehensive Final exam (paper and computer based). The distribution of grades is subject to some revision at the discretion of the instructor. Typical weighting values are given below:

Projects

Project 1

In this project, you are going to figure out the day of the week for three given input integers. User will enter three integers to specified as year, month and date. For the valid date, the user should give the exact day, for invalid input, the user should give an invalid remind. Check the sample runs below for more information. Your program should give the exact outputs for these runs. project_1.pdf.

Project 2

In this project, you are going to write a word search puzzle solver. Your program will read a 20x20 2D character array as input and a list of 30 words to search for. You will search the given words in the puzzle. Words can be any mix of uppercase and lowercase. The words may reside in vertical, horizontal, or diagonal directions, either left-right, up-down or vice versa. project_2.pdf.

Project 3

In this project, you will compare the similarity between images based on the histogram, which is a commonly used image-content based search. The similarity is based on difference of two histograms of a pair of images. project_3.zip and project_3_bonus.zip.
