EL-GY 9183: Advanced 3D Computer Vision: Data-driven 3D Object Processing - Spring 2016

Undergraduate Course, Abu Dhabi Campus

Spring 2016



3D computer vision (CV) is about recovery of structure or object/camera poses in 3D space. The structure from motion (SfM), simultaneous localization and mapping (SLAM) and camera/object pose estimation (3D rotations + 3D translations) as used in augmented reality (AR) and automatic panorama stitching algorithms are part of 3D computer vision. The system normally involves: image acquisition, feature detection, feature matching and model fitting. The 3D CV systems thus normally feeds from 1 (which as to be in motion in order to recovery the third dimension), 2 (stereo) or more camera systems. It can be designed to track easily localizable points such as markers on an actorâ€™s face in motion capture (mocap) for animations in movies and video games. Upon successful completion of this course, students will be able to: 1. Describe the representation of 3D image data. 2. Develop an efficient 3D object search engine. 3. Develop techniques for 3D object registration and matching. 4. Develop methods for robust 3D object recognition and segmentation.

Course Overview

3D object processing is an emerging field in computer vision with many applications across areas as diverse as engineering, science and medicine. The advancement in 3D acquisition technology has led to dramatic increase in the size of 3D datasets. We are therefore faced with a dramatic demand for automatic 3D model processing, understanding and analyzing techniques. Traditional works for 3D object processing are focused on the development of handcrafted 3D object feature, which dealt well with individual 3D object or a small-scale 3D object collection. In contrast, data-driven 3D object processing approaches are developed through aggregating the shape information from a collection of 3D objects, reasoning about key features and properties that are common in objects, and exploiting the relationships of objects. This course will introduce students to the techniques of data- driven 3D object processing. The focus of this course will be recent data-driven 3D object processing for 3D shape matching, retrieval, registration, recognition, segmentation, classification and clustering.

Intended Learning Outcomes:


  • Describe the representation of 3D image data.
  • Develop an efficient 3D object search engine.
  • Develop techniques for 3D object registration and matching.
  • Develop methods for robust 3D object recognition and segmentation.
  • Apply knowledge of 3D image data analysis techniques to analyze and model engineering problems to meet desired needs.

Prerequisites: EL 9143 (3D Computer Vision) or equivalents (discuss with instructor), and proficiency in programming (Matlab is required and C++ is optional)

Grading

Teaching and Learning Methodologies

Students are expected to arrive at class with an understanding of the basic definitions, concepts, and applications of relevant topics. Class time will be devoted to in- depth discussions of course topics. Project assignments are dedicated for reinforcing course topics via solving representative problem sets and holding class discussions.

Grading

Students are expected to understand the basic definitions, concepts, and applications of relevant topics in class. Grades will be based on four mini projects, one final project, reportand presentation. Project will be assigned after the completion of each specific topic. Projects

Project 1

In this project, you are asked to transform the given source data to best match the target model by implementing Iterative Closest Point (ICP) Algorithm in Matlab. project_1.pdf.

Project 2

In this project, you are going to design a shape search engine on dataset TOSCA to retrieve relevant 3D shapes for given query in Matlab. The dataset contains a total 147 objects, including 9 cats, 6 centaurs, 11 dogs, 21 gorillas, 17 horses, 15 lionesses, 3 wolves, 6 seahorses, 24 female figures and two different male figures, containing 15 and 20 poses. project_2.pdf.

Project 3

In this project, you are going to match pairs of corresponding points among given models (mesh015.off and mesh054.off) by implementing Hungarian algorithm. project_3.pdf.

Project 4

In this project, you are going to implement 3D shape segmentation on given 3 centaur models with different poses (centaur1.off, centaur2.off and centaur3.off). project_4.pdf.
